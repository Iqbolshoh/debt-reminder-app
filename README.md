# 💰 Debt Reminder System

A modern **Debt Tracking & Reminder System** built with **HTML, CSS, JavaScript, and PHP**.  
It helps users manage debts, track clients, and send reminder messages.

---

## 🚀 Features

✨ Modern dashboard UI (glassmorphism design)  
➕ Add new debt entries  
📋 View debt list (Paid / Pending status)  
📨 Send reminder messages via PHP backend  
📱 Responsive design (mobile friendly)  
🎨 Clean senior-level UI/UX design  
⚡ Fast lightweight vanilla stack

---

## 📂 Project Structure
<pre>
debt-reminder-app/
│── index.html      # Frontend UI
│── style.css       # UI Styling
│── script.js       # Frontend logic
│── send.php        # Backend handler
│── log.txt         # Message logs (auto created)</pre>

## 🧑‍💻 How It Works
Enter debtor name
Enter phone number
Enter debt amount
Click Send Reminder
PHP backend processes request
Message is logged in log.txt
